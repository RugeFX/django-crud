# Models / Migrations

## Creating a model

1. Go to an app (or create a new one with `python manage.py startapp {appname}`)
2. Create some model classes
2. Run `python manage.py makemigrations {appname}` to create a migration file for the specified app's models

## Applying migrations

`python manage.py migrate`

## Creating a superuser

`python manage.py createsuperuser`

## Putting models into the admin panel

Add `admin.site.register(ModelName)` to the admin.py file in the app
